# chardoncs.dev

Created with [`sv`](https://github.com/sveltejs/cli).


## Install dependencies 

```bash
bun install
```

## Development

Run a development environment.

```bash
bun --bun run dev
```

## Building

```bash
bun run build
```

> You may need to install an [adapter](https://svelte.dev/docs/kit/adapters) for your target environment.
